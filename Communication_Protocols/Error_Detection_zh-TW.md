## 🎯 **概述**

錯誤偵測與處理是可靠嵌入式通訊系統的關鍵組成部分。它們確保資料完整性、系統可靠性，以及在雜訊、干擾和硬體故障存在時的穩健運作。理解錯誤偵測方法和處理策略對於設計穩健的嵌入式系統至關重要。

### **關鍵概念**
- **錯誤偵測方法** - 同位元檢查、校驗和、CRC、錯誤更正碼
- **錯誤處理策略** - 錯誤回報、恢復機制、容錯
- **資料完整性** - 確保資料的準確性和一致性
- **系統可靠性** - 在錯誤條件下維持系統運作
- **效能影響** - 在錯誤偵測與系統效能之間取得平衡

### **面試官意圖（他們在探測什麼）**
- 你能否為給定的錯誤率選擇適合的錯誤偵測方法？
- 你是否理解偵測與更正之間的取捨？
- 你能否解釋錯誤如何在協定中傳播？

## 🧠 **概念優先**

### **偵測 vs 更正**
**概念**: 錯誤偵測識別錯誤，錯誤更正可以修復錯誤。
**為什麼重要**: 偵測更快且更簡單，更正更穩健但增加複雜度和額外開銷。
**最小範例**: 比較 8 位元同位元檢查（僅偵測）與漢明碼（偵測 + 更正）。
**試試看**: 實作兩種方法並測量效能和可靠性。
**重點**: 根據你的錯誤率和效能需求來選擇。

### **錯誤機率 vs 額外開銷的取捨**
**概念**: 更穩健的錯誤偵測方法會增加額外開銷，但能捕捉更多錯誤。
**為什麼重要**: 在嵌入式系統中，你必須在可靠性與效能及資源限制之間取得平衡。
**最小範例**: 比較 1KB 資料封包的校驗和與 CRC-32。
**試試看**: 測量不同錯誤偵測方法的效能影響。
**重點**: 將錯誤偵測強度與你的應用需求相匹配。

## 🤔 **什麼是錯誤偵測？**

錯誤偵測是識別嵌入式系統中資料傳輸、儲存或處理過程中發生的錯誤的過程。它涉及各種技術和演算法，旨在偵測資料損壞、傳輸錯誤和系統故障，確保可靠的運作和資料完整性。

### **核心概念**

**錯誤來源：**
- **傳輸錯誤**: 資料傳輸過程中發生的錯誤
- **儲存錯誤**: 資料儲存過程中發生的錯誤
- **處理錯誤**: 資料處理過程中發生的錯誤
- **硬體錯誤**: 由硬體故障或失靈引起的錯誤

**錯誤類型：**
- **位元錯誤**: 個別位元錯誤和損壞
- **框架錯誤**: 框架格式和結構錯誤
- **時序錯誤**: 時序和同步錯誤
- **系統錯誤**: 系統層級的錯誤和故障

**錯誤偵測方法：**
- **同位元檢查**: 使用同位元位元的簡單錯誤偵測
- **校驗和**: 使用校驗和演算法的錯誤偵測
- **CRC**: 用於穩健錯誤偵測的循環冗餘校驗
- **錯誤更正碼**: 進階的錯誤偵測與更正

## 💻 **實作**

### **基本錯誤偵測實作**

**同位元實作：**
```c
// 同位元檢查實作
typedef struct {
    uint8_t data;
    uint8_t parity;
} Parity_Data_t;

// 計算偶同位元
uint8_t calculate_even_parity(uint8_t data) {
    uint8_t parity = 0;
    for (int i = 0; i < 8; i++) {
        if (data & (1 << i)) {
            parity ^= 1;
        }
    }
    return parity;
}

// 檢查偶同位元
bool check_even_parity(Parity_Data_t* parity_data) {
    uint8_t calculated_parity = calculate_even_parity(parity_data->data);
    return calculated_parity == parity_data->parity;
}
```

**校驗和實作：**
```c
// 校驗和實作
typedef struct {
    uint8_t* data;
    uint16_t length;
    uint16_t checksum;
} Checksum_Data_t;

// 計算簡單校驗和
uint16_t calculate_checksum(uint8_t* data, uint16_t length) {
    uint16_t checksum = 0;
    for (uint16_t i = 0; i < length; i++) {
        checksum += data[i];
    }
    return checksum;
}

// 驗證校驗和
bool verify_checksum(Checksum_Data_t* checksum_data) {
    uint16_t calculated_checksum = calculate_checksum(checksum_data->data, checksum_data->length);
    return calculated_checksum == checksum_data->checksum;
}
```
